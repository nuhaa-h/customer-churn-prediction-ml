# customer-churn-prediction-ml
# Customer Churn Prediction Using Machine Learning

## Project Overview
This project predicts whether a telecom customer will leave the company or stay, using the Telco Customer Churn dataset.

## Dataset
Dataset: Telco Customer Churn Dataset  
Source: Kaggle  
Records: about 7043 customers  
Attributes: customer information, account details, services, charges, and churn status.

Target column: `Churn`
- Yes: customer left
- No: customer stayed

## Project Stages
1. Data Understanding
2. Data Cleaning and Preprocessing
3. Train/Test Splitting
4. Feature Analysis
5. Classification Models
6. Model Evaluation
7. Model Comparison

## Models Used
- Decision Tree Classifier
- Support Vector Machine (SVM)

## Evaluation Metrics
- Accuracy
